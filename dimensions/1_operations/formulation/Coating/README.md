# Coating

## Description

Application of a thin coating layer onto particles or tablets by spraying a coating solution in a controlled environment.

## Example

```xml
<Coating>
  <CoreMaterial>Tablet cores (500 mg)</CoreMaterial>
  <CoatingMaterial>HPMC-based film coating</CoatingMaterial>
  <CoatingMethod>Pan coating</CoatingMethod>
  <SprayRate>8 g/min</SprayRate>
  <Temperature>45 Â°C</Temperature>
  <Duration>90 min</Duration>
  <Equipment>Perforated coating pan</Equipment>
  <SafetyMeasures>Dust extraction, temperature monitoring</SafetyMeasures>
</Coating>
```
