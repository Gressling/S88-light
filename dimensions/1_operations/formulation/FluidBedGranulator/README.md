# FluidBedGranulator

## Description

Wet granulation in a fluidised bed by spraying a binder solution onto suspended powder particles.

## Example

```xml
<FluidBedGranulator>
  <FeedMaterial>Microcrystalline cellulose / API blend</FeedMaterial>
  <BinderSolution>PVP K30 in water (5 %w/w)</BinderSolution>
  <AirFlowRate>80 m³/h</AirFlowRate>
  <InletTemperature>60 °C</InletTemperature>
  <SprayRate>15 g/min</SprayRate>
  <Duration>45 min</Duration>
  <GranuleSize>300 µm (d50)</GranuleSize>
  <Equipment>Glatt GPCG-5</Equipment>
  <SafetyMeasures>Explosion-proof design, filter integrity test</SafetyMeasures>
</FluidBedGranulator>
```
