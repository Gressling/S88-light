# PAT

## Description

Process Analytical Technology: in-line or on-line analytical measurement for real-time process monitoring.

## Example

```xml
<PAT>
  <AnalyticalMethod>NIR spectroscopy</AnalyticalMethod>
  <MeasurementFrequency>Every 30 s</MeasurementFrequency>
  <Duration>120 min</Duration>
  <Equipment>In-line NIR probe</Equipment>
  <SafetyMeasures>Laser safety class 1</SafetyMeasures>
</PAT>
```
