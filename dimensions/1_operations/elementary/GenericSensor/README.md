# GenericSensor

## Description

Generic sensor reading that acquires measurement data from a specified instrument at defined intervals.

## Example

```xml
<GenericSensor>
  <DeviceId>TEMP-201</DeviceId>
  <SensorType>Thermocouple Type K</SensorType>
  <MeasurementInterval>5 s</MeasurementInterval>
  <Duration>120 min</Duration>
</GenericSensor>
```
