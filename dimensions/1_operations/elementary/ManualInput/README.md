# ManualInput

## Description

Prompts the operator for manual data entry or confirmation during process execution.

## Example

```xml
<ManualInput>
  <Prompt>Enter observed crystal colour</Prompt>
  <InputType>String</InputType>
  <DefaultValue>White</DefaultValue>
  <Timeout>300 s</Timeout>
</ManualInput>
```
