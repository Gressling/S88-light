# Centrifugation

## Description

Separation of heterogeneous mixtures by applying centrifugal force to exploit density differences between phases.

## Example

```xml
<Centrifugation>
  <Sample>Cell suspension</Sample>
  <Speed>4000 rpm</Speed>
  <Time>15 min</Time>
  <Temperature>4 degC</Temperature>
  <Pressure>1 atm</Pressure>
  <RotationDirection>Clockwise</RotationDirection>
  <Equipment>Refrigerated benchtop centrifuge</Equipment>
  <SafetyMeasures>Balanced rotor, lid locked</SafetyMeasures>
</Centrifugation>
```
