# Dry

## Description

Removal of residual moisture or solvent from a solid product by thermal or vacuum drying.

## Example

```xml
<Dry>
  <Product>Filter cake</Product>
  <DryingMethod>Vacuum oven</DryingMethod>
  <DryingTemperature>60 degC</DryingTemperature>
  <DryingTime>480 min</DryingTime>
  <Humidity>5 %RH</Humidity>
  <AirflowRate>0 L/min</AirflowRate>
  <Equipment>Vacuum drying oven</Equipment>
  <SafetyMeasures>Inert atmosphere, temperature limit</SafetyMeasures>
</Dry>
```
