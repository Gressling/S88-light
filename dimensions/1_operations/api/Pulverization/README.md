# Pulverization

## Description

Size reduction of solid material by mechanical grinding, milling, or crushing to achieve a target particle size.

## Example

```xml
<Pulverization>
  <FeedMaterial>Dried crystal cake</FeedMaterial>
  <ParticleSizeRequirement>50 um (d90)</ParticleSizeRequirement>
  <PulverizationEquipment>Ball mill</PulverizationEquipment>
  <PulverizationTime>30 min</PulverizationTime>
  <PulverizationSpeed>300 rpm</PulverizationSpeed>
  <CoolingOrHeating>Cryogenic (liquid N2)</CoolingOrHeating>
  <DustCollection>HEPA filter extraction</DustCollection>
  <SafetyMeasures>Dust mask, ear protection</SafetyMeasures>
</Pulverization>
```
