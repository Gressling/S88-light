# Evaporation

## Description

Removal of solvent from a solution by vaporisation, concentrating the dissolved solute.

## Example

```xml
<Evaporation>
  <Product>Crude extract in ethanol</Product>
  <EvaporationMethod>Rotary evaporation</EvaporationMethod>
  <EvaporationTemperature>40 degC</EvaporationTemperature>
  <EvaporationTime>60 min</EvaporationTime>
  <Solvent>Ethanol</Solvent>
  <Concentration>95 % removal</Concentration>
  <HeatingSource>Water bath</HeatingSource>
  <Equipment>Rotary evaporator, 1 L</Equipment>
  <SafetyMeasures>Bump trap, cold finger condenser</SafetyMeasures>
</Evaporation>
```
