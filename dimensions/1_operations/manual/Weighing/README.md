# Weighing

## Description

Gravimetric determination of the mass of a substance using an analytical or precision balance.

## Example

```xml
<Weighing>
  <Substance>Salicylic acid</Substance>
  <Amount>5.00 g</Amount>
  <BalanceType>Analytical balance (0.1 mg)</BalanceType>
  <Duration>2 min</Duration>
  <SafetyMeasures>Anti-static measures, tared vessel</SafetyMeasures>
</Weighing>
```
