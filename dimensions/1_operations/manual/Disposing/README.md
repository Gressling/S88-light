# Disposing

## Description

Collection, classification, and disposal of chemical waste generated during an experiment.

## Example

```xml
<Disposing>
  <WasteType>Halogenated organic solvent</WasteType>
  <DisposalMethod>Collect in designated waste container</DisposalMethod>
  <Amount>500 mL</Amount>
  <Duration>5 min</Duration>
  <Environment>Waste storage area</Environment>
  <SafetyMeasures>Labelled container, secondary containment</SafetyMeasures>
</Disposing>
```
