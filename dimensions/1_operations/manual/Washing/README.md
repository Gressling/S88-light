# Washing

## Description

Removal of impurities from a solid or liquid product by repeated contact with a wash solvent.

## Example

```xml
<Washing>
  <MaterialToBeWashed>Crude crystal cake</MaterialToBeWashed>
  <WashingSolventOrMedium>Cold deionised water</WashingSolventOrMedium>
  <WashingTemperature>5 degC</WashingTemperature>
  <WashingTime>10 min</WashingTime>
  <NumberOfWashingSteps>3</NumberOfWashingSteps>
  <WashingVolumeOrRatio>2:1 v/w</WashingVolumeOrRatio>
  <AgitationOrMixing>Gentle swirling</AgitationOrMixing>
  <Drying>Air dry on filter</Drying>
  <Equipment>Buchner funnel</Equipment>
  <SafetyMeasures>Gloves, eye protection</SafetyMeasures>
</Washing>
```
