# Setup

## Description

Preparation and assembly of equipment and instrumentation prior to running an experiment.

## Example

```xml
<Setup>
  <ExperimentName>Aspirin synthesis batch 042</ExperimentName>
  <SetupInstructions>Assemble reflux condenser, calibrate thermocouple, check stirrer</SetupInstructions>
  <Duration>20 min</Duration>
  <Equipment>Round-bottom flask, reflux condenser, heating mantle</Equipment>
  <SafetyMeasures>Inspect glassware for cracks</SafetyMeasures>
</Setup>
```
