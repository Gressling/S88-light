# GeneralCleaning

## Description

Post-process cleaning of equipment and work area after completion of an experiment or batch.

## Example

```xml
<GeneralCleaning>
  <Materials>Detergent, deionised water, isopropanol</Materials>
  <Procedure>Disassemble, soak, scrub, rinse, dry</Procedure>
  <Duration>30 min</Duration>
  <Environment>Wash bay</Environment>
  <SafetyMeasures>Gloves, apron</SafetyMeasures>
</GeneralCleaning>
```
