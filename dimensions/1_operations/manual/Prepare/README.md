# Prepare

## Description

Preparation of raw materials and reagents required for the subsequent process steps.

## Example

```xml
<Prepare>
  <Chemical>Salicylic acid</Chemical>
  <Chemical>Acetic anhydride</Chemical>
  <Duration>10 min</Duration>
  <Equipment>Analytical balance, spatula</Equipment>
  <SafetyMeasures>Gloves, lab coat</SafetyMeasures>
</Prepare>
```
