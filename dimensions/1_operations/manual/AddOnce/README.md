# AddOnce

## Description

Single addition of a defined quantity of a chemical reagent to the reaction vessel.

## Example

```xml
<AddOnce>
  <Chemical>Acetic anhydride</Chemical>
  <Timing>t = 0 min</Timing>
  <RateOfAddition>5 mL/min</RateOfAddition>
  <Temperature>25 degC</Temperature>
  <Pressure>1 atm</Pressure>
  <pH>7.0</pH>
  <Solvent>None</Solvent>
  <ReactionEnvironment>N2 blanket</ReactionEnvironment>
  <Equipment>Dropping funnel</Equipment>
  <SafetyMeasures>Fume hood, slow addition for exotherm control</SafetyMeasures>
</AddOnce>
```
