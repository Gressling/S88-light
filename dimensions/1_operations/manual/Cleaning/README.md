# Cleaning

## Description

Pre-process cleaning of equipment and glassware to remove contaminants before an experiment.

## Example

```xml
<Cleaning>
  <Materials>Acetone, deionised water</Materials>
  <Procedure>Triple rinse with solvent, then water</Procedure>
  <Duration>15 min</Duration>
  <Environment>Fume hood</Environment>
  <SafetyMeasures>Gloves, eye protection</SafetyMeasures>
</Cleaning>
```
