The material dimension declares chemicals, solvents, and reagents used in a recipe. Materials are referenced from operations via the `<Chemical>` element.

```xml
<Chemical>
    <Name>Acetic anhydride</Name>
    <Amount unit="mL">30</Amount>
    <CAS>108-24-7</CAS>
    <MolWeight unit="g/mol">102.09</MolWeight>
    <Role>Reagent</Role>
</Chemical>
```
